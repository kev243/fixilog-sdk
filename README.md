# Fixilog SDK

**Fixilog SDK** is the official client library for sending application errors to the Fixilog platform.  
It is lightweight, zero-config and built for modern JavaScript & TypeScript applications.

Fixilog allows you to:

- Capture errors from the browser, Node.js, or Next.js.
- Send structured error reports to your Fixilog dashboard.
- Let AI analyze, summarize and suggest fixes for every error (Fixilog AI).
- Receive real-time alerts via Slack, Discord or email.
